PR Tool 操作手冊

登入後，請點選 Tab 選擇組別。
如下圖為例，選到Capston 1 tab，之後點選 `Join Capstone 1`

加入後，`Join Capstone 1`按鈕會變成`New Pull Request`按鈕
發 PR 請點`New Pull Request`按鈕

點擊`New Pull Request`按鈕後的畫面。
操作方式：請點擊紅框處或拖拉檔案方式，上傳 Test Case 測試文件。
※ 前提是要建立好 branch 並且 branch 名稱要跟 Test Case 測試文件相同才行。
※ Ex：branch 命稱為 TMD-13246；Test Case 檔名為 TMD-13246

選擇完檔案，會進入 Pull Request 畫面。

需要自行填寫的東西有：
➢ 填寫 Reviewers、Assignees 以及 Labels ( 僅限第一次使用需要填寫，後續建立只要確認即可 ! )
➢ 填寫 Base on 哪個分支
➢ CI Result 連結

Base on TMD-XXXX 注意：
紅框 1 請輸入 (base on) TMD-XXXX，紅框 2 則會自動更改在紅框所選的分支，但還是請確認一下是否正確。

測試步驟檔案匯入內容會在這裡：

全部確認完成請按下`Create Pull Request`按鈕，就建立完成囉。
